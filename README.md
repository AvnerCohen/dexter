# Dexter

A library for logging.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     dexter:
       github: luckyframework/dexter
   ```

2. Run `shards install`

## Usage

```crystal
require "dexter"
```
